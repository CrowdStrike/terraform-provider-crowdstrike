# crowdstrike_ngsiem_data_connectors (Data Source)

List the available CrowdStrike Next-Gen SIEM data connectors, and optionally resolve a connector's ID by its exact name (for use as `connector_id` on a `crowdstrike_ngsiem_data_connection`).

## API Scopes

The following API scopes are required:

- NGSIEM Data Connections API | Read


## Example Usage

```terraform
terraform {
  required_providers {
    crowdstrike = {
      source = "registry.terraform.io/crowdstrike/crowdstrike"
    }
  }
}

provider "crowdstrike" {
  cloud = "us-2"
}

# List every available Next-Gen SIEM data connector.
data "crowdstrike_ngsiem_data_connectors" "all" {}

# Resolve a specific connector's ID by its exact name (use as connector_id on a data connection).
data "crowdstrike_ngsiem_data_connectors" "hec" {
  by_name = "HEC / HTTP Event Connector"
}

output "all_connectors" {
  value = data.crowdstrike_ngsiem_data_connectors.all.connectors
}

output "hec_connector_id" {
  value = data.crowdstrike_ngsiem_data_connectors.hec.id
}
```

## Schema

### Optional

- `by_name` (String) Exact connector name to match (e.g. `HEC / HTTP Event Connector`). When set, `id` is populated with that connector's ID and the lookup errors if no connector matches.

### Read-Only

- `connectors` (Attributes List) All available Next-Gen SIEM data connectors. (see [below for nested schema](#nestedatt--connectors))
- `id` (String) ID of the connector matched by `by_name`. Null when `by_name` is not set.

<a id="nestedatt--connectors"></a>
### Nested Schema for `connectors`

Read-Only:

- `description` (String) Connector description.
- `id` (String) Connector ID.
- `name` (String) Connector display name.
- `parsers` (List of String) Preset parser package name(s) for this connector. Generic connectors (for example HEC and Cribl) return an empty list; for those you set the parser yourself on the data connection.
- `type` (String) Connector type: `PUSH` (the source pushes data to an ingest URL; manageable with `crowdstrike_ngsiem_data_connection`) or `PULL` (CrowdStrike fetches from the source).
- `vendor_name` (String) Vendor name.
- `vendor_product_name` (String) Vendor product name.
